# my_first_react_project

This project is a small Movie Application built using **React** and the **openmdAPI**. It helps users search for and explore movie details, providing an intuitive interface and seamless user experience.

## Highlights

- Developed with **React** and **CSS**
- Ideal for learning about API integration and frontend development
- Hosted on **Netlify** for fast and reliable deployment
