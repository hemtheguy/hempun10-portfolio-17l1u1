# my_first_react_project

This is a small Movie Application built using React and openmdAPI. The project showcases movies with information fetched via an API, presenting clean and interactive user interface design. It was developed to enhance my React and CSS skills.

## Highlights

- Built with **React** and **CSS**
- Implements **openmdAPI** for fetching movie details
- Features responsive UI with smooth navigation
